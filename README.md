# Django_fc_hs
Python应用，使用django来展示，和播放数据！

环境Python3,Django1.11.7

可以本地搭建玩玩，不建议部署上线，原因你懂得！

huang_se -->  项目目录     

huangse --> App目录

migrations --> 数据库脚本生成目录

paurl --->爬虫所在目录

static -->静态文件目录

templates -->模板文件目录

huangsedb.sql --> sql数据

把/huang_se/huang_se/settings.py里面的这个数据库信息改成你自己的，并且在数据库里面创建好这个数据库！
